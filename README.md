# 🧠 ウェブで動く自作アキネーターシステム

ブラウザ上で動作する **自分用アキネーター（質問推測）システム** です。  
データ構造を定義でき、**フロントエンド主体**で軽量に動作することを目的としています。

---

## ✨ 特徴

- 🌐 **ブラウザだけで動作**
- 📄 **JSON / ローカルデータ対応**
- 🧩 **アキネーターデータを簡単に自作可能**
- ⚡ **Vite + Svelte による高速開発**
- 🔌 **サーバー依存を最小限に設計**
- 💕 **オープンソースでご自由にどうぞ**

---

## 🧠 システム概要

このアプリは以下の流れで動作します。

1. 📂 利用するアキネーターデータを選択  
2. ❓ 質問を順番に表示  
3. ✅ ユーザーの回答をもとに候補を絞り込み  
4. 🎯 最終的な推測結果を表示  

> データ構造を工夫することで  
> 「キャラ当て」「作品当て」「設定診断」などにも流用可能です。
